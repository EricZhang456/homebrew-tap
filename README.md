# EricZhang456 Tap

## How do I install these formulae?

`brew install ericzhang456/tap/<formula>`

Or `brew tap ericzhang456/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "ericzhang456/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
